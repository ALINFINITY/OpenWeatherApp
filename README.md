# App Clima Mundial 2026

Aplicacion web para consultar el clima en tiempo real de las ciudades sede del Mundial 2026.

Selecciona un pais y una ciudad para ver las condiciones climaticas actuales: temperatura, humedad, viento y mas.

## Tecnologias

- React
- TypeScript
- Vite
- OpenWeatherMap API

## Instalacion

```bash
npm install
npm run dev
```

## Variables de entorno

Crea un archivo `.env` con tu API key de OpenWeatherMap:

```
VITE_WEATHER_API_KEY=tu_api_key
```
